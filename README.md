# GraphicsEngine

Graphics Engine

A Cross Api Game engine Currently Supporting:
D3D11 
OpenGL 4.5

It supports:
Physics using Nvidia's PhysX

Both Renderers
Diffuse Lighting
Texture Mapping
Shadows mapping (static and dynamic)

OpenGL Supports
UI Rendering
Text Rendering
Grass system (instanced Rendering)
Particle system
Parallax Mapping
Normal Mapping
